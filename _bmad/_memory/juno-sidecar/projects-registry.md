## Usage

- **Create new:** Use `[GS] Genesis` or `[NP] New Project`
- **Switch:** Use `[LP] List Projects` then select by number
- **Register existing:** Use `[RP] Register Project` and provide path

---

## Notes

Projects are stored in subfolders under the working directory. Each project follows the standard structure:

```
/{project-name}/
├── 00-genesis/
├── 01-world/
├── 02-characters/
├── 03-plot/
├── 04-outline/
├── 05-drafts/
├── 06-editing/
└── _meta/
```
