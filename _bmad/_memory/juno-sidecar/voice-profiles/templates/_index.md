# Voice Profile Template Library

Voice templates are pre-built writing DNA profiles extracted from classic authors in the public domain. Load one as a starting point for your project's voice, blend elements from multiple templates, or use them as reference when building your own profile from scratch. Each template captures sentence architecture, rhythm patterns, narrative techniques, and stylistic signatures.

**Total templates: 35**
